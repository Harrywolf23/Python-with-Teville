## Getting Started

Clone the repository:

```bash
git clone https://github.com/Harrywolf23/Python-with-Teville.git
cd Python-with-Teville
```

Run Python files from the repository root:

```bash
python filename.py
```

If your system uses `python3`, run:

```bash
python3 filename.py
```
